# 背景图片文件夹

请将以下背景图片放置在此文件夹中：

## 需要的背景图片：

1. **主页背景** - `home-bg.jpg` 或 `home-bg.png`
   - 深蓝紫色渐变背景
   - 包含宇宙/科技感元素
   - 建议尺寸：1920x1080 或更高

2. **多元兼容页面背景** - `compatibility-bg.jpg` 或 `compatibility-bg.png`
   - 深色径向渐变背景
   - 建议尺寸：1920x1080 或更高

3. **免费开源页面背景** - `opensource-bg.jpg` 或 `opensource-bg.png`
   - 深蓝紫色背景
   - 建议尺寸：1920x1080 或更高

4. **FAQ页面背景** - `faq-bg.jpg` 或 `faq-bg.png`
   - 深紫色渐变背景
   - 建议尺寸：1920x1080 或更高

## 图片格式要求：
- 支持格式：JPG, PNG, WebP
- 建议使用 WebP 格式以获得更好的压缩效果
- 确保图片质量清晰，适合高分辨率显示

## 使用说明：
背景图片将通过CSS的 `background-image` 属性应用到各个页面。如果不需要自定义背景图片，系统将使用CSS渐变作为默认背景。
