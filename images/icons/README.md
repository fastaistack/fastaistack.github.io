## 图标要求：
- **颜色**：白色或浅色，适合深色背景
- **风格**：简洁现代，与整体设计风格一致
- **格式**：优先使用SVG格式，支持缩放且文件小
- **质量**：高清，适合高分辨率显示

## 使用说明：
所有图标都通过HTML的 `<img>` 标签引用，确保图标文件路径正确。如果某些图标缺失，系统会显示占位符或使用默认样式。
